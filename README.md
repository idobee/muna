## 📌 About Muna

**뮤나(Muna)**는 *Music*과 *Luna*의 합성어로, 음악의 세계를 달처럼 환하게 비추는 AI 에이전트입니다.  
유튜브 음악 채널을 데이터 기반으로 자율 운영하며, 트렌드 스캔부터 콘텐츠 업로드까지 모든 과정을 자동화합니다.
